catimg is a little program written in C with no dependencies that
prints images in terminal. It supports JPEG, PNG and GIF formats.
.
This program was originally a script that did the sameby using ImageMagick convert.
.
You will need a 256 color terminal in order to use this program. For higher
resolution images, you need unicode support.
