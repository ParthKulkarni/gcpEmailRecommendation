I wonder if you could force the installer to run in a lower resolution,
which would then hopefully be scaled up.
Adding something like 'video=1024x768' or such to the command line when
booting the installer might work.
I suppose it would be handy if the fb in the installer somehow read
the dpi of the screen and could switch to a larger font if it exceeded
some reasonable value.  Certainly running with a 16x32 Terminus font
selected in console-setup makes my fb console very nice on a 15.6"
2880x1800 display.  The installer when I setup this machine was pretty
painful I must admit.
