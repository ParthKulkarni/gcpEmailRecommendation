Provides the Bio::Tools::Run::Phylo::PAML::* modules which provide an
interface to the PAML suite of programs for phylogenetic analyses of
DNA or protein sequences using maximum likelihood.
This package is part of the Bioperl project, a coordinated effort to
collect computational methods routinely used in bioinformatics into a
set of standard CPAN-style, well-documented, and freely available Perl
modules.
