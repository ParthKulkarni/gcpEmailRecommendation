Ukwm is a small window manager, using GTK+ and Clutter to do everything. It's
fork from mutter, and made some adjustments for ukui desktop environment, such
as add the support of "alt-tab" and adjust the UI.
.
This package contains the window manager shared library, used by ukwm itself.
And this package will be maintained by the Kylin Team.
