CryptoMiniSat is a modern SAT solver. The actual cryptominisat binary
links against the cryptominisat shared library that comes with headers
for C/C++ and a Python module.
CryptoMiniSat is an optional dependency of SageMath which has already
been packaged by the debian-science team. I will contact the
debian-science-sagemath list for advice and sponsors for this package.
