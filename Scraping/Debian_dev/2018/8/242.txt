  Description     : X2Go Session Broker for managing Linux/Windows terminal server farms
 X2Go Session Broker is an X2Go component that provides
 session brokerage for X2Go Servers via HTTP(S) and/or SSH.
 .
 Broker client support is provided by X2Go Client and Python X2Go alike.
 .
 Furthermore, the X2Go Session Broker is able to mimick the UCCS protocol
 originally used by Canonical's UCCS portal and works nicely together with
 Arctica Greeter's remote logon feature.
 .
 This package will be maintained under the umbrella of the Debian Remote
 Team.
