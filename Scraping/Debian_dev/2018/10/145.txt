Theme-D-Gnome is a software library providing GNOME bindings for
programming language Theme-D. It allows you to make GUI's in Theme-D.
The library is bases on guile-gnome-2 library. I have recently uploaded
Theme-D to mentors, too. The library contains an example program
implementing a calculator for X Windows.
I plan to maintain Theme-D-Gnome myself. I need a sponsor for the package.
