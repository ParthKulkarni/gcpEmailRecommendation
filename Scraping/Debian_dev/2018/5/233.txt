Description: Quick JavaScript Documentation generator
Docco is a quick documentation generator, written in Literate
CoffeeScript. It produces an HTML document that displays your comments
intermingled with your code. All prose is passed through Markdown, and
code is passed through Highlight.js syntax highlighting.
