 "Convenient parsing for Fetch."
 .
 The Fetch standard defines requests, responses, and the process that
 binds them: fetching.
 .
 This module provides convenient parsing on top of Fetch. It has
 built-in support for parsing JSON, CSV and TSV.
 .
 This module also allows the user to parse additional formats by using
 the `text` function directly (details available in API reference). This
 module replaced d3-request which provided a convenient alternative to
 XMLHttpRequest.
 .
 Node.js is an event-based server-side JavaScript engine.
d3-fetch is a dependency for d3 module and ultimately the 
blackst0ne-mermaid module.
