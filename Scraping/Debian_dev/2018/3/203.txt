This package is an optional dependency of lexicon, a DNS manipulation library
currently being packaged in Debian and used for Letsencrypt automation (bug
number #888800).
PyNamecheap is a Namecheap API client in Python, which supports:
* Registering a domain
* Checking domain name availability
* Listing domains you have registered
* Getting contact information for a domain
* Setting DNS info to default values
* Set DNS host records
