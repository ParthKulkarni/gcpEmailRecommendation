Deepin's qt5integration provides a library plugin used by Deepin
application and Deepin Desktop Enviroment. It implements many extra
features on top of stock Qt, include hijacking window decoration /
shadow painting, better support for cursors under HiDPI, retrieval of
full list of windows under current workspace, support for window of
arbitrary shape with anti-aliasing and so on.
I intend to co-maintain this package under pkg-deepin team.
