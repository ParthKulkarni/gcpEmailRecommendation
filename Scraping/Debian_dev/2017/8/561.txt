RTS2, or Remote Telescope System, 2nd Version, is an open source
observatory manager. Its job is to make an astronomical observatory
observing and user happy with the data gathered by the instruments.
It is hosted on github [1]. It will maintained within the Debian
Astronomy Working Group. A git repository will be created on alioth
[2].
