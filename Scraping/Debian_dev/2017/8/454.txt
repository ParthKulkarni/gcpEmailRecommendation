SkyPat is a C++ performance analyzing and testing framework. Programmers
only need to write test cases via SkyPat, and SkyPat helps to analyze
the programs' correctness and performance.
Currently, SkyPat provides many performance information for analysis,
such as cache miss, context switch, cpu cycle and instruction.
I intend to maintain this package and SZ Lin agreed to sponsor me.
