  Description     : ZeroMQ binding for Octave
ZeroMQ is a library which extends the standard socket interfaces, 
providing an abstraction of asynchronous message queues, multiple 
messaging patterns, message filtering (subscriptions), seamless access 
to multiple transport protocols and more.  This package contains the 
binding to the ZeroMQ library for use in Octave, a scientific 
computation software.
This package adds original functionality to Octave, allowing programs 
written in this language to communicate with programs written in several 
of other languages (C, C++, C#, Java, Lua, Perl, PHP, Python, Ruby, etc) 
through the powerful socket mechanism provided by the ZeroMQ library.  It 
will be maintained in the realm of the Debian Octave Group.
