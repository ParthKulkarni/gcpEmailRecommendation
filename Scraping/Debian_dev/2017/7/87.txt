Bootbox.js is a small JavaScript library which allows one to create
programmatic dialog boxes using Bootstrap modals, without having to
worry about creating, managing or removing any of the required DOM
elements or JavaScript event handlers. Bootstrap modals are dialog
prompts provided by the Bootstrap JavaScript framework.
The janus-demos package is currently in contrib because libjs-bootbox is
not yet packaged for Debian. Packaging libjs-bootbox gets us one step
closer to moving janus-demos into main.
