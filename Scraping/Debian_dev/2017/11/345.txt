  Description     : power stress and shaping tool
 Power Stress and Shaping Tool (PSST) is a controlled power tool
 for Intel SoC components such as CPU and GPU. PSST enables very
 fine control of stress function without its own process overhead
 This tool is useful for kernel developers on x86 systems when
 debugging power/scheduler characteristics to see if the CPUs are
 functioning correctly with respect to power CPU freq scaling
 controls.
