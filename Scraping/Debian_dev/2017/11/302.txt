Libcleri is a powerful tool to build languages. From a built language, libcleri
can automatically create parse trees, which are data structures representing
how a grammar matches input. It also provides feedback in case the input does
not match the language. This can be useful for auto-completion, suggestions or
error handling.
This package is a dependency of SiriDB, another ITP that I'll file shortly.
I'll start a team to maintain the SiriDB suite once alioth is enabled to create
